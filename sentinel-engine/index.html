Six security gates — all must pass before any transaction is signed
Gate 1
Domain age check
Sites under 30 days old are hard-blocked. Phishing sites are almost always on fresh domains. 30–90 days triggers a warning.
Gate 2
Lookalike detection
PhishGuard compares every URL against the full DefiLlama protocol domain list using similarity scoring. Catches “uniswop.io” style clones.
Gate 3
GoPlus contract scan
Every token contract scanned for honeypots, sell restrictions, hidden mints, blacklist functions, and proxy contract concealment.
Gate 4
Tenderly simulation
Transaction dry-run before any signing. Every asset change inspected. Unexpected outflow from a burner wallet triggers immediate abort.
Gate 5
Permit() detection
All six known permit() variants blocked at calldata level before nonce allocation. The number one drainer technique, stopped before it starts.
Gate 6
Gas ceiling
Any transaction requiring over 500,000 gas is rejected. Legitimate airdrop claims never exceed this. Gas-guzzler drainers always do.
